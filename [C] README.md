# Lending Club Loan Default Predictor

Interactive Streamlit app showcasing ML-powered credit risk prediction on 2.26M real Lending Club loans.

**Live demo:** *(add Streamlit Cloud link after deploying)*

---

## Quick Start (Local)

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Export model + EDA data from the cleaned dataset
python setup_model.py --data /path/to/lending_club_cleaned.csv

# 3. Run the app
streamlit run app.py
```

After step 2, you'll have `model/` and `data/` folders with everything the app needs.

---

## Deploy to Streamlit Cloud

1. Push this `streamlit-app/` folder to a GitHub repo (or add it to your existing `lending-credit-analysis` repo under a `streamlit/` subfolder).

2. **Include the data files.** Before pushing, run `setup_model.py` locally so the `model/` and `data/` folders are populated. Commit them to the repo.

3. Go to [share.streamlit.io](https://share.streamlit.io), sign in with GitHub, and create a new app:
   - **Repository:** `ngwanelegacie/lending-credit-analysis`
   - **Branch:** `main`
   - **Main file path:** `streamlit/app.py` (or wherever you placed it)

4. Hit **Deploy**. Streamlit Cloud reads `requirements.txt` automatically.

## Features

- **Predict Risk** — Enter loan details, get a default probability with risk badge (LOW / MEDIUM / HIGH)
- **Explore Data** — Interactive Plotly charts: default rates by grade, purpose, term, ownership, and time
- **Model Performance** — Side-by-side comparison of Logistic Regression, Random Forest, and Gradient Boosting + feature importance
